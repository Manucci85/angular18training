Application Médicale

Charte d'utilisation

Afin de vous offrir un suivi médical optimal et de permettre à votre médecin de communiquer avec vous via cette application, vous devez accepter la présente charte d'utilisation.

Cette application stocke vos données personnelles et médicales conformément aux lois européennes et belges sur la protection des données. Ces données sont utilisées uniquement pour la continuité de vos soins et ne seront jamais partagées sans votre consentement.

Vous pouvez à tout moment :

  • Consulter les données vous concernant
  • Faire une demande de suppression de vos données

En acceptant cette charte, vous autorisez :

  • Votre médecin à utiliser cette application pour communiquer avec vous et suivre vos données médicales.
  • L'application à stocker vos données personnelles et médicales dans le respect des lois applicables.

Pour plus d'informations, veuillez consulter :

  • Le texte complet du RGPD
  • Les directives belges sur le RGPD
  • Les lois sur le traitement des données médicales en Belgique
